def pipython.interfaces.gcsdll.get_dll_path (   dllname)

Return absolute path to GCS DLL as string.

Search for GCS DLL in this order: 'dllname' is absolute path to GCS DLL. If not, return GCSTranslator path from registry/sysenv. If not set return current working directory.

Parameters
dllname: Name or path to GCS DLL.
