Device connection

Here you will find sample code for how to connect one ore more PI devices. For further reading see the python scripts in the samples subdirectory in the provided ZIP file.

Connect a single device via the GCS DLL

By a dialog

The GCS DLL provides a graphical user interface to select the interface and parameters.

1 from pipython import GCSDevice
2 with GCSDevice('C-884') as gcs:
3  gcs.InterfaceSetupDlg()
4  print('connected: {}'.format(gcs.qIDN().strip()))

If you pass an optional key as arbitrary string the DLL remembers the settings in the Windows registry and will recall them the next time you connect with the same key.

1 from pipython import GCSDevice
2 with GCSDevice('C-884') as gcs:
3  gcs.InterfaceSetupDlg('MyTest')
4  print('connected: {}'.format(gcs.qIDN().strip()))

By a dedicated interface

You can connect over these interfaces with the according functions.

  • RS-232: ConnectRS232(comport, baudrate)
  • USB: ConnectUSB(serialnum)
  • TCP/IP: ConnectTCPIP(ipaddress, ipport=50000)
  • TCP/IP: ConnectTCPIPByDescription(description)
  • NI GPIB: ConnectNIgpib(board, device)
  • PCI board: Connect(board)
1 from pipython import GCSDevice
2 with GCSDevice('C-884') as gcs:
3  gcs.ConnectTCPIP('192.168.178.42')
4  print('connected: {}'.format(gcs.qIDN().strip()))

The parameter serialnum can be the serial number of the device as string or the device identification returned by EnumerateUSB. For TCP/IP ConnectTCPIP connects by given IP address where ConnectTCPIPByDescription uses the string from EnumerateTCPIPDevices.

By the device identification

There are functions to scan for available devices.

  • USB: EnumerateUSB(mask='')
  • TCPIP: EnumerateTCPIPDevices(mask='')

Use the mask to limit the number of found devices. It is a string that must be part of the identification string - see qIDN - returned by the devices.

1 from pipython import GCSDevice
2 with GCSDevice('C-884') as gcs:
3  devices = gcs.EnumerateTCPIPDevices(mask='C-884.4DB')
4  for i, device in enumerate(devices):
5  print('{} - {}'.format(i, device))
6  item = int(input('Select device to connect:'))
7  gcs.ConnectTCPIPByDescription(devices[item])
8  print('connected: {}'.format(gcs.qIDN().strip()))

Connect daisy chain devices

You have to open the interface once and than you connect all devices to this interface. Each device must have a unique ID on the daisy chain (see controller manual). There must be one device with ID 1 which needs not to be the master device (i.e. connected to the PC). See an example for 3 devices on an RS-232 daisy chain.

  • C-863 controller with device ID 3, this is the master device
  • E-861 controller with device ID 7
  • C-867 controller with device ID 1

There is no need to close the connections. This is done automatically because GCSDevice is used as context manager.

1 from pipython import GCSDevice
2 with GCSDevice('C-863.11') as c863:
3  c863.OpenRS232DaisyChain(comport=1, baudrate=115200)
4  # c863.OpenUSBDaisyChain(description='1234567890')
5  # c863.OpenTCPIPDaisyChain(ipaddress='192.168.178.42')
6  daisychainid = c863.dcid
7  c863.ConnectDaisyChainDevice(3, daisychainid)
8  with GCSDevice('E-861') as e861:
9  e861.ConnectDaisyChainDevice(7, daisychainid)
10  with GCSDevice('C-867') as c867:
11  c867.ConnectDaisyChainDevice(1, daisychainid)
12  print('\n{}:\n{}'.format(c863.GetInterfaceDescription(), c863.qIDN()))
13  print('\n{}:\n{}'.format(e861.GetInterfaceDescription(), e861.qIDN()))
14  print('\n{}:\n{}'.format(c867.GetInterfaceDescription(), c867.qIDN()))

Low level interface

Usually you connect with GCSDevice via the GCS DLL. But on platforms where the GCS DLL is not available you can connect using a socket.

1 from pipython.gcscommands import GCSCommands
2 from pipython.gcsmessages import GCSMessages
3 from pipython.interfaces.pisocket import PISocket
4 gateway = PISocket(host='192.168.178.42', port=50000)
5 messages = GCSMessages(gateway)
6 gcs = GCSCommands(messages)
7 print gcs.qIDN()
8 gateway.close()

Unknown devices

When you call GCSDevice with the controller name the according GCS DLL is chosen automatically. For unknown devices you can specify a dedicated GCS DLL instead.

1 from pipython import GCSDevice
2 with GCSDevice(gcsdll='PI_GCS2_DLL.dll') as gcs:
3  gcs.InterfaceSetupDlg()
