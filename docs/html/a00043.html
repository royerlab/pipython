def pipython.gcscommands.getbitcodeditems (   value,
  allitems = None,
  items = None 
)

Return boolean dictionary of 'items' according to corresponding bit in 'value'.

If 'allitems' is None the number of bits in 'value' is used.

Parameters
value: Integer of the bit mask.
allitems: Single item or list or None. Length must not match the bit width of 'value'.
items: Item or list or None. Only these items are returned. If None then 'allitems' are returned.
Returns
: Ordered dictionary {items: value} where value is True or False.

Split 'answer' into item/values dict of according types.

Parameters

valueconv: List of conversion functions for values. If there are more values than conversion functions the last given conversion function is used for the remaining values.
